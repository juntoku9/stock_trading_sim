# Project Goal

PaperTrade Pro is a frontend-first stock trading simulator designed to let users practice equity trading without real money.

The product goal is to provide a realistic but low-risk learning environment where a user can:
- create a profile and join a public or private league,
- start with fixed virtual capital,
- track live or simulated stock price movement,
- buy and sell stocks to build a portfolio,
- monitor net worth and trade history over time,
- compare performance on a leaderboard,
- learn core investing concepts inside the app,
- and use an AI assistant for market context and educational guidance.

In short, the app should help users learn market mechanics, test trading decisions, and understand portfolio performance through a polished virtual simulation, not real financial execution.
